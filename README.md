# Todo App (React + Custom Hooks)

Более продвинутое приложение списка задач, созданное для практики React и архитектуры с кастомными хуками.

##  Что реализовано

- Разделение интерфейса на компоненты
- Переиспользуемые UI компоненты
- Управление состоянием через `useState`
- Добавление / удаление задач
- Поиск задач
- Сохранение задач в `localStorage`
- Автофокус на input после действий
- Скролл к первой невыполненной задаче
- Подтверждение удаления всех задач
- Оптимизация производительности (`useMemo`, `useCallback`)

---

##  Кастомные хуки

### `useTasks`
Главный хук приложения — вся бизнес-логика:

- хранение задач
- добавление / удаление
- переключение статуса
- поиск задач
- работа с input
- сохранение в localStorage
- оптимизация через `useMemo` и `useCallback`

---

### `useTasksLocalStorage`
Отвечает за работу с localStorage:

- загрузка задач при старте
- сохранение при изменении

---

### `useIncompleteTaskScroll`
Логика скролла:

- находит первую невыполненную задачу
- хранит ref для скролла

---

##  Структура компонентов

- **Field** — универсальное поле ввода
- **Button** — кнопка
- **AddTaskForm** — добавление задачи
- **SearchTaskForm** — поиск задач
- **TodoInfo** — количество задач + Delete All
- **TodoItem** — отдельная задача
- **TodoList** — список задач

---

##  Используемые хуки React

- `useState` — состояние
- `useEffect` — побочные эффекты
- `useRef` — доступ к DOM (фокус, скролл)
- `useMemo` — оптимизация вычислений
- `useCallback` — оптимизация функций
