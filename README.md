# Todo App (React)

Простое приложение списка задач, сделанное для практики React.

## Что реализовано

- Разделение интерфейса на компоненты
- Переиспользуемые UI компоненты
- Базовая структура React приложения

## Структура компонентов

- **Field** — универсальное поле ввода
- **Button** — кнопка с текстом
- **AddTaskForm** — форма добавления новой задачи
- **SearchTaskForm** — форма поиска задач
- **TodoInfo** — блок с количеством задач и кнопкой Delete All
- **TodoItem** — отдельная задача
- **TodoList** — контейнер списка задач
